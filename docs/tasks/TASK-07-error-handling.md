# TASK-07: 오류 처리 구현

## 설명
잘못된 입력 및 계산 불가 상황에 대한 오류 메시지를 구현한다.

## 관련 요구사항
- NFR-06: 오류 메시지 표시

## 오류 유형 및 메시지

| 상황 | 오류 메시지 |
|------|------------|
| 0으로 나누기 | "오류: 0으로 나눌 수 없습니다" |
| 음수의 제곱근 | "오류: 음수의 제곱근은 계산할 수 없습니다" |
| 0 이하의 로그 | "오류: 0 이하의 값은 로그를 계산할 수 없습니다" |
| tan 정의 불가 각도 | "오류: 정의되지 않는 값" |

## 구현 내용
- [ ] 오류 메시지를 결과 영역에 표시 (빨간색/경고색)
- [ ] 오류 발생 시 히스토리에 기록하지 않음
- [ ] 오류 상태에서 AC/Backspace로 정상 복귀 가능
- [ ] 각 오류 상황별 메시지 표시 테스트
