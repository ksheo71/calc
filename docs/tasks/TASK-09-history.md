# TASK-09: 히스토리 구현

## 설명
계산 히스토리를 표시하고, 이전 결과를 재사용하는 기능을 구현한다. 앱 재시작 후에도 히스토리가 유지된다.

## 관련 요구사항
- FR-14: 히스토리 표시
- FR-15: 히스토리 재사용
- FR-16: 히스토리 전체 삭제

## 구현 내용
- [ ] 히스토리 목록 UI 영역 구현
- [ ] 계산 완료 시 히스토리 목록 최상단에 항목 추가 (`수식 = 결과` 형식)
- [ ] 최소 20개 항목 저장 및 스크롤 지원
- [ ] Electron `localStorage` 또는 파일 시스템으로 히스토리 영구 저장
- [ ] 히스토리 항목 클릭 시 결과값을 입력란에 채우기
- [ ] 히스토리 전체 삭제 버튼 구현
- [ ] 오류 결과는 히스토리에 추가하지 않음
