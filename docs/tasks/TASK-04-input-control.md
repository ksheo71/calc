# TASK-04: 입력 제어 구현

## 설명
AC(전체 초기화), Backspace(마지막 문자 삭제), 키보드 입력을 구현한다.

## 관련 요구사항
- FR-23: AC (전체 초기화)
- FR-24: Backspace (마지막 문자 삭제)
- NFR-04: 키보드 입력 지원

## 구현 내용
- [ ] AC 버튼 클릭 시 수식/결과 초기화 (메모리 유지)
- [ ] Backspace 버튼 클릭 시 마지막 문자 삭제
- [ ] 빈 입력란에서 Backspace 오류 없이 처리
- [ ] 키보드 이벤트 리스너 등록
  - 숫자 키 (0~9), 소수점 (.)
  - 연산자 키 (+, -, *, /)
  - Enter → = (계산 실행)
  - Backspace → 마지막 문자 삭제
  - Escape → AC (전체 초기화)
