# TASK-10: 메모리 기능 구현

## 설명
M+, M-, MR, MC 메모리 기능을 구현한다.

## 관련 요구사항
- FR-17: M+ (메모리 더하기)
- FR-18: M- (메모리 빼기)
- FR-19: MR (메모리 불러오기)
- FR-20: MC (메모리 초기화)

## 구현 내용
- [ ] 메모리 변수 초기화 (0)
- [ ] M+ 버튼: 현재 표시값을 메모리에 더하기
- [ ] M- 버튼: 현재 표시값을 메모리에서 빼기
- [ ] MR 버튼: 메모리 값을 입력란에 불러오기
- [ ] MC 버튼: 메모리 값 0으로 초기화
- [ ] 메모리에 값이 있을 때 `M` 인디케이터 표시
- [ ] 메모리 값이 0일 때 `M` 인디케이터 숨김
- [ ] AC로 메모리 초기화되지 않는 것 확인
- [ ] 단위 테스트 작성
