# TASK-03 검증방법: 화면 표시

## 검증 목표
입력 수식과 계산 결과가 화면에 올바르게 표시되는지 확인한다.

## 사전 조건
- TASK-01, TASK-02 완료

## 수동 검증 (UI)

### 수식 표시 확인
1. 앱 실행
2. 숫자 `1`, `2`, `3` 순서로 클릭 → 수식 영역에 `123` 표시 확인
3. `+` 클릭 → `123 +` 표시 확인
4. `4`, `5`, `6` 클릭 → `123 + 456` 표시 확인

### 결과 표시 확인
5. `=` 클릭 → 결과 영역에 `579` 표시 확인

### 긴 수식 스크롤 확인
6. 매우 긴 수식 입력 → 수식 영역이 스크롤되거나 줄임 처리되는지 확인

### 지수 표기 확인
7. `9`, `^`, `9`, `9` 입력 후 `=` → 결과가 지수 표기법(`e+`)으로 표시되는지 확인
