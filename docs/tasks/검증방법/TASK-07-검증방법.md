# TASK-07 검증방법: 오류 처리

## 검증 목표
모든 오류 상황에서 올바른 메시지가 표시되고, 정상 복귀가 가능한지 확인한다.

## 사전 조건
- TASK-01~06 완료

## 수동 검증 (UI)

### 오류 메시지 확인

| 조작 | 기대 결과 |
|------|----------|
| `1`, `÷`, `0`, `=` | "오류: 0으로 나눌 수 없습니다" (빨간색) |
| `-1`, `√` | "오류: 음수의 제곱근은 계산할 수 없습니다" (빨간색) |
| `0`, `log` | "오류: 0 이하의 값은 로그를 계산할 수 없습니다" (빨간색) |
| DEG 모드에서 `90`, `tan` | "오류: 정의되지 않는 값" (빨간색) |

### 오류 후 복귀 확인
1. 위 오류 중 하나 발생
2. AC 클릭 → 정상 초기화 확인
3. 새로운 계산 정상 수행 확인

### 히스토리 미기록 확인
4. 오류 발생 후 히스토리 목록에 오류 항목이 없는지 확인
