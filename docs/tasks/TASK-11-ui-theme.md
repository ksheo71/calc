# TASK-11: UI/테마 구현

## 설명
라이트 테마를 적용하고, 버튼 색상 구분 및 시각적 피드백을 구현한다.

## 관련 요구사항
- NFR-02: 라이트 테마

## 구현 내용
- [ ] 라이트 테마 CSS 작성 (흰색/밝은 회색 배경)
- [ ] 버튼 색상 구분
  - 숫자 버튼: 흰색/회색 계열
  - 연산자 버튼 (+, -, ×, ÷): 주황색 계열
  - 공학 함수 버튼: 파란색 계열
  - = 버튼: 녹색 계열
  - AC/Backspace: 빨간색/회색 계열
- [ ] 버튼 hover/active 시각적 피드백 구현
- [ ] 텍스트와 배경 명암 대비 확보
- [ ] 오류 메시지 빨간색/경고색 표시
- [ ] `M` 인디케이터 스타일 구현
- [ ] DEG/RAD 인디케이터 스타일 구현
