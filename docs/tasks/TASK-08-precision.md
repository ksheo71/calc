# TASK-08: 부동소수점 정밀도 처리

## 설명
계산 결과의 부동소수점 오류를 최소화하고, 소수점 10자리 이내로 표시한다.

## 관련 요구사항
- NFR-05: 부동소수점 정밀도

## 구현 내용
- [ ] 부동소수점 반올림 함수 구현 (소수점 10자리)
- [ ] `0.1 + 0.2 = 0.3` 정확히 표시되도록 처리
- [ ] 소수점 10자리 초과 시 반올림 처리
- [ ] 매우 큰 수(1e15 이상) 지수 표기법 변환
- [ ] 매우 작은 수(1e-10 이하) 지수 표기법 변환
- [ ] 단위 테스트 작성
