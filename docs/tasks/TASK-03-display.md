# TASK-03: 화면 표시 구현

## 설명
입력 수식과 계산 결과를 화면에 표시하는 기능을 구현한다.

## 관련 요구사항
- FR-21: 입력 수식 화면 표시
- FR-22: 계산 결과 화면 표시

## 구현 내용
- [ ] 수식 표시 영역 UI 구현 (상단)
- [ ] 결과 표시 영역 UI 구현 (하단)
- [ ] 숫자/연산자 입력 시 수식 영역 실시간 업데이트
- [ ] 계산 완료 시 결과 영역 업데이트
- [ ] 수식이 화면 너비 초과 시 스크롤 처리
- [ ] 지수 표기법 표시 처리 (매우 크거나 작은 수)
