# FR-10: 제곱근 (√)

## 설명
입력값에 대해 제곱근(√) 함수를 계산한다. 음수에 대해 오류를 처리한다.

## 입력
- 피연산자 A (숫자, 0 이상)

## 출력
- √A 의 결과값
- A < 0 인 경우: 오류 메시지

## 수용 기준
- [ ] √0 = 0 이 계산된다
- [ ] √4 = 2 이 계산된다
- [ ] √2 ≈ 1.4142135623 이 계산된다
- [ ] √(-1) 입력 시 "오류: 음수의 제곱근은 계산할 수 없습니다" 메시지가 표시된다
- [ ] 계산 결과가 FR-22에 따라 화면에 표시된다
- [ ] 계산 후 결과가 FR-14에 따라 히스토리에 기록된다

## 관련 요구사항
- FR-22: 계산 결과 화면 표시
- FR-14: 히스토리 표시
- NFR-05: 부동소수점 정밀도
- NFR-06: 오류 메시지 표시
